      □ Thanks to Philippe Carl, an ROI selected in the ROI Manager is

        method now sets the magnification and is more robust (
        JavaScript example).

        interface.

        converted to binary using custom thresholding methods (example
        ).

        and Convert to Mask commands.

        imageCalculator() macro function to not select the first
        (output) image in batch mode macros.

        be updated after updating an existing plot window using the
        Plot.update() macro function.

        when converting selections into masks.

        to not correctly open 16-bit, 4 samples per pixel TIFFs, such
        as those created by VueScan.
