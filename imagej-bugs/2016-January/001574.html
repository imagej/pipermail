--- Comment #1 from Mark Hiner <hinerm at gmail.com> ---
Hello,

This "PROTECTED_LOCATION" is because your Fiji is installed in Program Files.
This can cause permissions problems with the file modifications necessary for
updating.

See http://imagej.net/Install_Fiji_on_Windows for more information. For now the
recommended solution is simply to move Fiji.app to a directory outside Program
Files (e.g. your user directory).
