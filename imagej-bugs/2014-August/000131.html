UI locks up completely, with menus, toolbar inaccessible and not possible to
work with the image, e.g. to make ROIs.

Example to trigger this behavior:

1 Launch Fiji
2 Use "n" to make a new image, accept defaults
3 Make a rectangular ROI in new, empty image
4 Enter a menu, e.g. "File", but do nothing
5 Click the image outside of the ROI to make the ROI go away
6 UI now locked up. Can be accessed again after right clicking the image

Repeat from 4 if UI is not locked up at 6, as it does not always lock up the
first time this is done after launching Fiji.

I get this behavior on my system if keyboard layout is Icelandic "is" or Swiss
German "ch", but not for "us". These are the only layout I have tried.
