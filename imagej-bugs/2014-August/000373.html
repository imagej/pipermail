--- Comment #20 from Wayne Rasband <wsr at nih.gov> ---
Upgrade to the latest ImageJ daily build (1.49h3) and you can work around this
bug by putting the macro tool in the macros/toolsets folder. Single macro tools
in the toolsets folder added to the toolbar using the toolbar's ">>" menu are
now restored to the toolbar when ImageJ restarts as long as the file name
matches the tool name and there is at least one free tool slot.
