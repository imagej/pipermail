--- Comment #17 from Wayne Rasband <wsr at nih.gov> ---
I noticed that Fiji runs the "Postage Action Tool" command when what it should
do is install the tool in the toolbar. In ImageJ, type "L", find the "Postage
Action Tool", run it, and it is installed in the toolbar. Do the same thing in
Fiji and it is not installed in the toolbar. Instead, "Tile" is displayed in
the Log window, which indicates Fiji is trying to run the tool as if it were an
ordinary macro. ImageJ installs the tool using the MacroInstaller#installTool()
function, which is called from Macro_Runner when the name of the macro file
ends in "Tools.ijm".
