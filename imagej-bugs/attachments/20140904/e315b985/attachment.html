Location of TIFF files

When I open a large TIFF (approx. 1.5 Gb) in ImageJ version 1.49g, the image is
garbled with color shifts and apparent separation and lateral shifts of RGB
channels.  I haven't tried opening the image with other versions of ImageJ.

The problem occurs with ImageJ2 data structures turned on or off.  With the
SCIFIO option turned on, I receive a Java error (separate bug report
submitted).

The large TIFF and a smaller TIFF are located in the link provided in the
attachment.  Both files were generated using PhotoScan Pro, a proprietary
photogrammetry program by Agisoft (agisoft.ru).  The smaller TIFF opens without
problems in ImageJ.</pre>
