        <pre>To reproduce:
 - open a czi file containing multiple z slices, using Bio-formats importer
with "Open as virtual stack"
 - Image > Transform > Rotate 90 Degrees Right
 - scroll through the stack
All z slices appear the same now, being copies of the slice that was active
when performing the rotation.

On multi-channel images, the channels are retained (i.e. different images), but
the z slices are lost (i.e. all copies of the same plane).

This bug seems to be specific for files opened via Bio-formats, since opening
tif files via Import > TIFF Virtual Stack and then performing rotation works as
expected.

Cheers,
Jan
