        <pre>Hi Sebastian,

The IMCF update site deploys a custom Stitching plugin, which clashes with the
plugin provided by the Bio-Formats update site.

By default, Fiji ships Bio-Formats 5.0.x. The Bio-Formats update site provides
Bio-Formats 5.1 which has breaking API changes that affect the Stitching
plugin. Thus the Bio-Formats update site also provides a 5.1-compatible
Stitching plugin.

Enabling IMCF overwrites the Bio-Formats update site's Stitching plugin with a
5.0 version, which then breaks because of 5.1 and 5.0 incompatibility.

To fix your installation, I would suggest uninstalling both Bio-Formats and
IMCF update sites ONE AT A TIME.

Then just enable one of the two based on your current needs (e.g. Bio-Formats
if you need 5.1).

I am also going to file a bug report for the updater, as I could not find a way
to have both update sites installed but have the Bio-Formats Stitching.jar
"win".</pre>
