slices, that there was a long lag time for the images to display. So there

MRI data. This macro now runs extremely slowly and is painful to use. 

I have read on the FIJI website that with the new updates there are issues with
Fiji and Java on MacOSX. I mainly use Mac and it is now mostly useless for my
purposes. If it would help I could upload data and the macro.
