        <pre>What command are you trying to run to convert your image?

an ImgPlus - which typically is only the case when you open an image with an
ImageJ2 command.

convert command? That should open as an ImgPlus. But be aware that when using a
command requiring an ImgPlus, you are using beta functionality that may not be
fully tested yet.</pre>
