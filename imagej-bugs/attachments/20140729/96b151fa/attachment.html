        <pre>When there is a macro (stored on a network location) with changes open in the

local drive everything goes well.

It seems reproducable, I tried different files and different locations, but I
can't completely exclude that there is another reason for the crash:

Start the macro editor (Plugins -> new -> Macro).
Open a macro that is stored on a network location.
Change something.
Close Fiji.
Click 'Yes' to save changes.
Fiji hangs.
