        <pre>Dear all,

as Curtis said, the bug does not spawn from the ScreenGrabber plugin, as it
simply uses the Robot class from base Java awt (as a proof of this, the snipped
area size is correct).
I was able to solve the problem in Linux Mint by turning off the visual effects
of Cinnamon. I got a hint from this Oracle bug report, which reports Robot not
working correctly with translucent windows:
<a href="http://bugs.java.com/view_bug.do?bug_id=7043845">http://bugs.java.com/view_bug.do?bug_id=7043845</a>

So you should be able to get what you want by simply disabling windows
translucency or any special effects the recent Linux windows managers (Gnome3,
Unity, Cinnamon) have forced upon us.

Federico</pre>
