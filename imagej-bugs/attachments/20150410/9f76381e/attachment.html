        <div>
        <pre>Some DeltaVision files (for example, see attached uploaded image
"2269_37C_12_R3D.dv" are read into FIJI in a way that the intensities are on a
scale of -32728 to 32767 (instead of 0 to 65535). This then leads to a problem
when trying to quantitate intensity values, for example, using multi-measure
through the ROI manager. This happens on both a Mac OS X and Windows 8 machine.
As it does not happen with all DeltaVision files, it may be an issue with how
DeltaVision is writing the files, but I don't know why that would be.</pre>
