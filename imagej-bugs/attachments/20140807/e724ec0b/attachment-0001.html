        <pre>When a file is opened, closed, and then re-opened, the file name (image title)
is not the original, but a sequentially numbered version instead, as if the
previous instace of the image remained open.

Example: 
test_file.tiff             -> first time it opens in session
test_file-1.tif          -> second time, after closing previous file
test_file-2.tif          etc.

The sequence continues normally when the several instances of same file are
opened simultaneously:

test_file.tiff             --> first time it opens in session
test_file-1.tif          --> second time, after closing previous file
test_file-2.tif          -->a second instance of the same image
test_file-3.tif          -->a new instance, after closing all previous ones
