picture: three-layers.png) I can try to convert it in a single layer RGB image,
via Image->Color->Stack to RGB. I get a color picture, but that is mess up (see
rgb-merge.png). 
If I export this image as a png, for example, and I open it with Gimp, it looks
normal. So there is a problem somewhere with the visualiziation of this image.
