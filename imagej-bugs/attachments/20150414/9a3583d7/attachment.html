type. For example, you can't replace a rectangular selection with a line
selection. This is because of this change made in ImageJ 1.49g:

   * Thanks to Philippe Carl, an ROI selected in the ROI 
   Manager is deselected when it is removed from the image.

Why do you need to replace a selection in the ROI Manager with a different type
