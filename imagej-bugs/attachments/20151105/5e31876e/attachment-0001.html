        <pre>Two issues -
1. dialog boxes that are blank except for the "ok" or other buttons
Even when starting FIJI, I get a blank dialog box because I have a duplicate
plugin. If I click outside of FIJI, the text in the box often appears. It
becomes confusing when I have many windows open and don't know what FIJI is
trying to communicate to me. This makes FIJI difficult to use (although I keep
trying) and I switch to ImageJ which normally works and I can read the dialog
boxes.

Interestingly even when trying to submit this bug report I got a blank dialog
box which I later found out told me the submission failed.

2. Sync Windows often gets a step behind.
When I have synced windows, often if drawing a rectangle in one window, it will
draw in the others but not the target window. Sometimes clicking outside of
FIJI corrects this. Often, particularly if my computer sleeps, the synced
windows command is very problematic. If it allows me to select a line or
rectangle, it seems to cause blank histograms, and blank profiles. It applies
the magnifying glass tool nonuniformly across the windows, apparently does not
allow use of selecting tools (rectangles, ellipses, lines...). Sync Windows is
problematic in FIJI and ImageJ.
