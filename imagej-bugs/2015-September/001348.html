I don't know what you are refering to by mentioning "folks" and "RE OSX", but
as mentioned on the ImageJ mailing list [1], there's a recent update that
breaks Fiji on all platforms.

A workaround to fix existing installations is to run the following command from
the command prompt:

    ImageJ-win64.exe --update update jars/imagej-deprecated.jar

See also: https://github.com/imagej/imagej-ops/issues/201
