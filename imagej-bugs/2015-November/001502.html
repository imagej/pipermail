I use the built-in macro function "getDirectory" for getting information about
where the images to be analyzed are located and this the line of code:

Folder = getDirectory("Choose the folder where your images are located");

This used to work beautifully, but since I upgraded my computer to Mac OS X El
Capitan (10.11.1) I no longer get the title text displayed. It still works
without any error messages, and the parameter "Folder" will have the correct
path information. But if you don't know how the macro works you might not know
that you should choose a folder and not a particular image file when the title
text is not displayed in the file open dialog. 

Best regards, Maria
