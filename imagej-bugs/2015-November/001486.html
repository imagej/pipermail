I think this issue started when I upgraded to El Capitan, but it may have been
when I updated to Java 8. In either case, there are a lot of little issues.
I've seen a few of them in the forum.

1) Clicking the FIJI icon in the dock opens up a new java instance. I can't
drag and drop files onto the instance.
2) metadata is no longer parsed correctly from our DV files (DeltaVision)
3) Often, when clicking from window to window I lose the all of the menu bar
disappears except for the ImageJ menu. It can take a while for it to come back. 
4) performance is very slow and seems to deteriorate over time. Sometimes it
will take 30s just to responde to changes using the brightness/contrast tool.
(opening the bug report form took ~1 min)
