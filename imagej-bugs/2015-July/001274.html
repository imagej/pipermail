--- Comment #1 from Mark Hiner <hinerm at gmail.com> ---
Note that Extended_Depth_Field ships with Fiji - you should not need to
manually install it.

"Install Plugins" is just a shortcut to copy a plugin-containing file to your
ImageJ.app/plugins directory. Since Extended_Depth_Field is already there in a
Fiji installation, it seems like you may have actually deleted or overwritten
the existing .jar.

As you can see in the system info attached to this bug report,
"Extended_Depth_Field" is marked "NOT_INSTALLED" in your system - which is why
restarting Fiji isn't helping: there is no jar to discover.

If you run the Updater[1] you should be able to re-install Extended_Depth_Field
from the Fiji update site. You may need to go to advanced mode if it doesn't
immediately show it as ready for installation.

Let us know if you have any problems re-installing from the update site, or if
any of this is unclear.

Thanks,
Mark


[1] http://imagej.net/Updater
