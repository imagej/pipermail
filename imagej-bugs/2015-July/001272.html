It is probably an issue with Java 8 on OS X. One option is to downgrade to Java
6 for the time being:

http://imagej.net/FAQ#How_do_I_set_up_Java_6_on_OS_X.3F

Once things get slow, you can also try to obtain information about why it is
slow, by following the instructions at:

http://imagej.net/Debugging#Debugging_JVM_hangs

This might help diagnose what is going wrong deep within Java, for example,
which when followed up with a targeted web search will give ideas for working
around it.
