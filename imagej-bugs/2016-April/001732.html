--- Comment #1 from Curtis Rueden <ctrueden at wisc.edu> ---
Things to try:

1. Launch ImageJ in debug mode:
http://imagej.net/Troubleshooting#Launching_ImageJ_from_the_console

2. Update to the very latest version of Java 8, if possible

3. Try a fresh download of Fiji bundled with Java 8: http://fiji.sc/#download

Note that Fiji is now bundled with Java 8 and has the Java-8 update site
enabled, which provides newer component versions; see
http://imagej.net/2015-12-22_-_The_road_to_Java_8

That said, I am pretty surprised by this crash regardless. In particular,
please try launching in debug mode as described above to see if you receive any
extra info on the command line after Fiji dies.

See also:
http://imagej.net/Troubleshooting#If_ImageJ_crashes
