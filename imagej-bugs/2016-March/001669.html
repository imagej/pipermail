the fact, that one of the most important PlugIns I use, OMERO connect plugin
requires Java 1.7 or higher. The version supplied with Fiji is 1.6, so I
disabled it forcing the programme to use the system version. This allows some
plugins to run, but the ones requiring Java 3D will not function. I had the IT
department install the new Java 3D, but the problem persists. the exact error
