Create a new image (8-bit, 400x400)

Activate the multi-point ROI tool.

Click on the image to add 2 or more points and they are placed in the corrrect
position.

Clear the points.

Zoom in so that the window shows only part of the image.

Click on the image. The first point is added correctly. The subsequent points
are not created in the correct location. 

Note that points can be moved to the correct location by dragging.

So the click-to-create function maps screen coordinates to the wrong place on
the image. Dragging a points maps coordinates correctly.
