--- Comment #1 from Wayne Rasband <wsr at nih.gov> ---
The 1.5GB TIFF is in 16-bit ARGB format, which the ImageJ TIFF reader does not
currently support. You should be able to work around this problem by having
PhotoScan Pro save in 16-bit RGB format. The smaller TIFF file opens correctly
because it is in 8-bit RGB format. It would with debugging if you could create
a smaller 16-bit ARGB TIFF in PhotoScan Pro and upload it.
