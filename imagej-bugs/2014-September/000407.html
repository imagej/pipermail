I think the problem has been solved by the suggestion from Mark Hiner:

"This is probably related to the ImageJ2/SCIFIO update.. you can temporarily
disable this by running the Edit>Options>ImageJ2 plugin and unchecking the
SCIFIO option."

I use several machines that have generated this error, and the last couple
(windows or mac)  of times I ran into this issue with subsequent updates, I
just turned the ImageJ2 plugin off and all worked fine (although still throws
up an results-with-error window sometimes).  So I think it's ImageJ2-related
and good to go until that gets fixed.
