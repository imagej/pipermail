The DeltaVision format is handled by Bio-Formats, which has their own system
for reporting issues. For instructions, see:
https://www.openmicroscopy.org/site/support/bio-formats5.1/about/bug-reporting.html

Also, there is a new version of Bio-Formats (5.1) that you can test by enabling
the Bio-Formats update site (http://fiji.sc/List_of_update_sites). This issue
may already be resolved.
