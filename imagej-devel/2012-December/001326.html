Hi all,

I cut the release for ImageJ 2.0.0-beta6 today, including ImgLib2 [1, 2],
SCIFIO [3, 4] and ImageJ2 [5, 6]. (It would have been ideal to use an
existing SCIFIO release rather than cutting one specifically for ImageJ2,
but the ImageJ2 codebase now uses SCIFIO's 4.5 development branch, and
there is no 4.5.x release yet.)

Later tonight or tomorrow I will update the ImageJ update site and web
site, and announce on the ImageJ user list.

Happy holidays, all!

Regards,
