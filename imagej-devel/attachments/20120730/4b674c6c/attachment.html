</div>Yes, the annotation processing is a little hairy. Our solution in ImageJ2<br>

which demand that annotation processors present in the classpath when<br>
compiling Java sources be run.<br>
<br>
I have ticket in the ImageJ2 ticket system to make it easier to start<br>

that situation and run the annotation processors manually.<br>
