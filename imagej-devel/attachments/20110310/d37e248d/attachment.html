Hi everyone,<br><br>Please note that the directory structure in SVN has been updated. Many directory names have been simplified, mostly by removing the &quot;ij-&quot; prefixes. (The project names, and hence JAR files, still have the &quot;ij-&quot; prefix.) If you are developing in Eclipse, it will probably be easiest to delete all ImageJDev-related projects and reimport them (Import &gt; Existing Maven Projects, or Import &gt; Check out Maven Projects from SCM). For details, see: <a href="http://imagejdev.org/ides">http://imagejdev.org/ides</a><br>

<br>It is likely that the directory structure and project names will be more stable now, as we have made good progress on the ImageJ2 project design in recent weeks.<br><br>-Curtis<br>
