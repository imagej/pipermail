Some ideas about the &quot;about&quot; image. In Fibonacci order:<br>
<br>
1. What about making this an open contest and advertise a bit mode widely it<br>
in the IJ and Fiji lists perhaps?, rather than IJ-devel only?<br>
We might otherwise be missing lots of interesting stuff.<br>
Also that would it make people feel a bit more interested  in the development<br>
of IJ2.<br>
<br>
2. Relax the theme and let&#39;s see what other images than cells people submit.<br>
<br>
3. I also think that we would want to see what the images would look like at<br>
the intended resolution of the about window, so I suggest the images should be<br>
submitted at that size, reduced or cropped to avoid disappointment when<br>
reducing them (and detail being lost) or scaling artifacts.<br>
Maybe this can be done in some of the wikis (the Tudor one has, I think, a<br>
plugin (?) for polls that might be possible to adapt to this?).<br>
<br>
5. There are lots of interesting and beautiful images for which it would be<br>
difficult or impossible to read any text overwritten on them, so an idea would<br>
be to use the images with an artificial darkening horizontal gradient or<br>
similar so the text is readable.<br>
<br>
8. I like the idea of various images alternating or randomly used for this, so<br>
perhaps there could be a pool of sample images to choose from at run time. If<br>
space is an issue, when the about samples folder is not existent or empty the<br>
about window could use just a default image stored in the jar file or a blank<br>
frame.<br>
