Dear all,

It's great to see IJ2 in beta. And extra-great that my legacy plugins
work there (at least the ones I have tested so far).

One small comment:
I use drag n' drop a lot, to open images and scripts, and to install
plugins. The default behaviour in IJ2-beta1 seems to be to open all
dragged-n-dropped files in a text window, which is not that handy unless
the file is actually a text file. Otherwise, plugins should get
installed, images should get opened as images and scripts should be
opened in the script editor.

Keep up the excellent work,

Michael
